# geo411-regression
R analysis of regression between SAT score and GPA and Mosquito Pupae and Mosquito Breeding Site
# Title : 
An Analysis of Regression Relationship between SAT Score and GPA, and Mosquito Pupae and Mosquito Breeding Site
# Overview: 
This is an analysis of 2 different situation 1)SAT score and GPA to see if there an regressional relationship between the two factors, and 2) number of mosquito pupae and number of mosquito development sites in Thailand. 
# Method: 
We use the summary command, an R command to get the basic summary and we use R Studio to produce Residuals vs Fitted Plots, Q-Q residuals plot, scale-location plots, and residuals vs leverage plots to determine the relationship.
# Key Findings:
There is an positive regressional relationship between SAT Score and GPA, although the nature of residuals vs fitted plot lead us to conclude SAT score and GPA are not independent of each other. There is also a positive relationship between numbers of mosquito pupae and mosquito breeding site.
# Tools:
R Studios, pen, paper, Microsoft Word, calculator and Github
